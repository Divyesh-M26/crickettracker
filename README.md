# 🏏 Cricket Tracker – Player Analytics Project

## 📌 Overview
Cricket Tracker is a Python-based data analysis project that processes cricket statistics and generates insights using graphs and calculations. It helps visualize player performance in a simple and meaningful way.

---

## 🚀 Features
- Load and process cricket dataset
- Player-wise statistics (runs, wickets, averages, strike rate)
- Graphical visualization using charts
- Easy comparison between players

---

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📊 What This Project Does
- Converts raw cricket data into meaningful insights
- Displays performance trends visually
- Helps understand player consistency
